# Lab 01 – User Login & Account Issues

## Scenario
User is unable to log in to their Windows 11 computer.

## Environment
- OS: Windows 11
- Account Type: Local User
- Tools Used: Computer Management, Settings

## Troubleshooting Steps
1. Logged in as administrator.
2. Opened Computer Management → Local Users and Groups.
3. Checked if the user account was disabled or locked.
4. Reset the user password.
5. Verified account status.
6. Tested user login.

## Resolution
User account was locked due to multiple failed login attempts.  
Password was reset and account access restored.

## Evidence
Screenshots included showing:
- User list
- Account lockout status
- Password reset confirmation
- Successful login
