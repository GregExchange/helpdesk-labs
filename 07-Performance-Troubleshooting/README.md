## Scenario
User reported the Windows 11 system was running slow.

## Objective
Identify performance bottlenecks and apply safe troubleshooting steps to improve system responsiveness.

## Environment
- OS: Windows 11 Home
- Tools Used: Task Manager, Startup Apps, Storage Settings, Windows Security

## Troubleshooting Steps
1. Verified user complaint by observing system slowdown with multiple apps running
2. Used Task Manager to check CPU, Memory, and Disk usage
3. Identified top resource-consuming processes
4. Ended safe non-system processes causing high usage
5. Disabled unnecessary startup applications
6. Checked storage space and cleaned temporary files if needed
7. Ran Windows Defender quick scan
8. Rebooted and confirmed improved performance

## Evidence
Screenshots included in `/screenshots` folder showing performance metrics before and after fixes.

## Outcome
System performance improved after reducing resource load and optimizing startup behavior.
