# Lab 05 – Software Installation & Removal

## Scenario
User requested a software install. Later, user requested removal.

## Objective
Install an approved application, verify installation, uninstall it, and verify complete removal.

## Environment
- OS: Windows 11 Home
- Tools Used: Settings (Installed apps), Downloads, Start Menu

## Software Used
- 7-Zip (x64)

## Steps Performed
1. Verified the app was not installed
2. Downloaded installer from official vendor site
3. Installed the software
4. Verified install via Installed apps + launching the app
5. Uninstalled the software
6. Verified removal

- Installation completed

- Application launched
- Uninstall process
- Verified removed

## Outcome
Software was installed successfully, then removed cleanly per user request.
