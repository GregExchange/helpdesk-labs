# Printer Issues – Help Desk Lab

## Scenario
User reports printer is not printing.

## Environment
- OS: Windows 11
- Printer: Microsoft Print to PDF
- Tools: Settings, Printers & Scanners

## Troubleshooting Steps
1. Verified printer installation.
2. Set printer as default.
3. Checked and cleared print queue.
4. Printed test page.

## Root Cause
Stuck print job in queue.

## Resolution
Cleared queue and resumed printing.

## Skills Demonstrated
- Printer troubleshooting
- Device management

## Outcome
Printer functioning normally.
# Lab 04 – Printer Issues

## Scenario
User reports printer is not printing.

## Objective
Diagnose and restore printer functionality on Windows 11.

## Environment
- OS: Windows 11 Home
- Printer: Microsoft Print to PDF
- Tools Used: Printers & Scanners, Services

## Troubleshooting Steps
1. Verified printer status
2. Simulated issue by pausing printer
3. Restarted Print Spooler service
4. Cleared stuck print queue
5. Resumed printer and tested printing

## Resolution
Printer resumed normal operation.

## Skills Demonstrated
- Printer troubleshooting
- Service management
- End-user support
