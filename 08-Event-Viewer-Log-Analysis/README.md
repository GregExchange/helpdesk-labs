# Lab 08 – Event Viewer Log Analysis

## Scenario
User reports frequent system errors.

## Objective
Identify and analyze system error logs using Windows Event Viewer.

## Environment
- OS: Windows 11 Home
- Tool Used: Event Viewer

## Steps Performed
1. Opened Event Viewer and navigated to System logs
2. Filtered logs for Error and Warning events
3. Analyzed event details including Source and Event ID
4. Cleared System logs and rebooted system
5. Verified reduced error events

## Evidence
Screenshots included showing error filtering, event analysis, and log clearing.

## Outcome
System errors were reviewed and logs were cleared for monitoring.

## Skills Demonstrated
- Event Viewer filtering
- Error analysis
- Root cause identification
- System log maintenance
